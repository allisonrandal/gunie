=head1 DESCRIPTION

Gunie is an implementation of the Perl 1.0 language, written in Go.

This code is distributed under the terms of the Artistic License 2.0.

=head1 PREREQUISITES

You'll need golang installed, together with the pigeon PEG parser
generator package. On both Debian/Ubuntu and Fedora/RHEL, golang can
be installed with the package 'golang'. Then pigeon can be installed
with:

  $ go get -u github.com/pointlander/peg
